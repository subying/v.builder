# vbuilder
==============
vbuilder is a front-end automation construction framework based on gulpJS.

## install
```
npm install vbuilder
```

## Usage

### Initiation
```
gulp init
```

### Development
```
gulp 
// or 
gulp dev 
// or
gulp --e dev
```

### Release
```
//Released to test environment
gulp --e test

//Released to pre production environment
gulp --e pre

//Release to production environment
gulp --e www
```
